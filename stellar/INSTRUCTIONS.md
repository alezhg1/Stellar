# 🚀 Stellar — Полная инструкция по запуску

## ✅ Шаг 1: Установка зависимостей

```bash
cd /workspace/stellar
npm install
```

**Что установится:**
- Next.js 14, React 18
- better-sqlite3 (локальная БД)
- uuid (генерация ID)
- framer-motion (анимации)
- tailwindcss (стили)
- И другие зависимости

---

## ✅ Шаг 2: Создание .env.local

```bash
cp .env.example .env.local
```

Заполните `.env.local`:

```env
# OpenRouter API ключ (бесплатно)
OPENROUTER_API_KEY=sk-or-v1-ваш-ключ

# Для локальной версии Supabase не нужен!
```

**Где получить ключ OpenRouter:**
1. Перейдите на https://openrouter.ai/keys
2. Зарегистрируйтесь (можно через Google)
3. Создайте новый ключ
4. Скопируйте и вставьте в `.env.local`

---

## ✅ Шаг 3: Инициализация базы данных

База данных создастся автоматически при первом запуске!

Файл `stellar.db` появится в корне проекта после запуска `npm run dev`.

**Что внутри:**
- Таблица `profiles` — пользователи
- Таблица `topics` — темы ФИПИ (10 тем математики)
- Таблица `user_topic_mastery` — прогресс пользователя
- Таблицы `chat_sessions` и `chat_messages` — история чата

---

## ✅ Шаг 4: Запуск проекта

```bash
npm run dev
```

Откройте браузер: **http://localhost:3000**

## 🎯 Как тестировать MVP

### 1. Онбординг
- Откройте http://localhost:3000/onboarding
- Введите имя, класс (8-11), экзамен (ОГЭ/ЕГЭ)
- Пройдите мини-диагностику (5 вопросов)
- Получите карту пробелов

### 2. Карта знаний
- Откройте http://localhost:3000/dashboard
- Посмотрите на цветовую индикацию тем
- Кликните на тему для деталей
- Обратите внимание на красные темы (критично)

### 3. AI-репетитор
- Нажмите "Спросить AI-репетитора" на теме
- Задайте вопрос: "Как решить уравнение 2x+5=13?"
- AI будет задавать наводящие вопросы, а не давать ответ
- Попробуйте ошибиться — AI объяснит ошибку

### 4. Прогноз баллов
- На дашборде посмотрите блок "Прогноз результата ЕГЭ"
- Обратите внимание на рекомендации
- See слабые темы и потенциальный прирост баллов

## 🔧 Технические детали

### Локальная БД (SQLite)
Вместо облачного Supabase используется файл `stellar.db`:
- ✅ Работает без интернета
- ✅ Не требует регистрации
- ✅ Быстрее для локальной разработки
- ❌ Нет синхронизации между устройствами

### AI-модель
Используется бесплатная модель через OpenRouter:
- Модель: `meta-llama/llama-3-8b-instruct:free`
- Лимит: ~100 запросов/день бесплатно
- Альтернатива: платный тариф от $5/мес

### Socratic Orchestrator
Системный промпт заставляет AI:
1. Не давать готовые ответы
2. Задавать наводящие вопросы
3. Хвалить за правильные шаги
4. Объяснять ошибки

## 🐛 Возможные проблемы и решения

### Ошибка: "Module better-sqlite3 not found"
```bash
npm rebuild better-sqlite3
```

### Ошибка: "OPENROUTER_API_KEY is required"
Проверьте, что `.env.local` создан и содержит ключ

### Ошибка: "Database is locked"
Закройте все процессы Node.js и удалите `stellar.db`:
```bash
rm stellar.db
npm run dev
```

### Чат не отвечает
Проверьте ключ OpenRouter и интернет-соединение
