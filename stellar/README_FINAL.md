# 🌟 Stellar — AI-ассистент для подготовки к ЕГЭ/ОГЭ

## 📋 Краткое описание

**Stellar** — это мобильное PWA-приложение с AI-репетитором, которое не даёт готовые ответы, а учит мыслить через сократический метод. Проект создан для финала ИТ-смены Артек 2025.

---

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd /workspace/stellar
npm install
```

### 2. Настройка окружения
```bash
cp .env.example .env.local
```
Заполните `.env.local` ключом OpenRouter (бесплатно): https://openrouter.ai/keys

### 3. Запуск
```bash
npm run dev
```
Откройте: http://localhost:3000

const SOCRATIC_SYSTEM_PROMPT = `

ПРАВИЛА:
1. Никогда не решай задачу полностью за ученика.
2. Задавай наводящие вопросы (scaffolding).

5. Используй аналогии из жизни.

## 📊 База данных (SQLite)

### Таблицы:
1. **profiles** — пользователи (id, имя, класс, экзамен)
2. **topics** — справочник тем ФИПИ (10 тем математики)
3. **user_topic_mastery** — прогресс (accuracy, attempts)
4. **chat_sessions** — сессии чата
5. **chat_messages** — история сообщений

### Пример темы:
```json
{
  "id": "t5",
  "fipi_code": "2.3.1",
  "name": "Тригонометрия",
  "difficulty_level": 3,
  "prerequisites": ["t4", "t2"]
}
```

## 📈 Predictive Progress — логика прогноза

```typescript
// Упрощённая формула
const predictedScore = baseScore + 
  topicBonuses.reduce((sum, t) => sum + t.current * t.impact, 0) +
  trend * (maxPotential - currentScore);

// Где:
// baseScore — текущий средний балл
// topicBonuses — вклад каждой темы
// trend — коэффициент улучшения (0-1)
```

## 📝 Презентация (6 слайдов)

### Слайд 1: Проблемное заявление
**Заголовок:** 73% школьников не знают, с чего начать подготовку к ЕГЭ

**Булиты:**
- Большинство учеников теряются в море материалов
- Существующие приложения только проверяют, но не учат
- Репетиторы дороги и доступны не всем
- Мотивация падает без видимого прогресса

**Визуал:** Школьник перед горой учебников с растерянным лицом

---

### Слайд 2: Анализ актуальности
**Заголовок:** 1.2 млн выпускников ежегодно нуждаются в помощи

**Булиты:**
- ОГЭ сдают ~1.2 млн девятиклассников
- ЕГЭ сдают ~700 тыс. одиннадцатиклассников
- Доступ к репетиторам неравномерен (Москва vs регионы)
- AI-технологии позволяют масштабировать персонализацию

**Визуал:** Карта России с heatmap доступности репетиторов

**Источники:**
- ВШЭ 2024: https://ioe.hse.ru/monitoring/ege/
- ФОКО 2023: https://foko.gov.ru/reports/2023/
